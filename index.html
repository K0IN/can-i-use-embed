Can I Use Embed API Examples

API Features:

  • Case-insensitive feature lookup: Works with WebGL, webgl, WEBGL, etc.
  • Flag support: Shows 'requires_flags' status for features that need special configuration
  • Multiple features: Combine multiple features to get the highest required version
  • Browser filtering: Filter results to specific browsers using the filter parameter
  • SVG output: All endpoints return SVG images for easy embedding
Available Endpoints:
  • /all-features - Get list of all available features (JSON)
  • /min-browser-version - Get minimum browser versions for features (SVG)
  • /supported-browsers - Get supported browsers for features (SVG)
  • /target-version - Target version analysis (coming soon)
